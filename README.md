# Student Data Cleaning and Analysis Project

This is a beginner-friendly Python portfolio project that demonstrates how to:
- clean a messy dataset
- handle missing values
- remove duplicates
- standardize text formatting
- create new useful columns
- generate basic visualizations
- write simple business-style insights

## Project Files
- `students_raw.csv` → messy dataset
- `clean_data.py` → cleans the dataset
- `analyze_data.py` → creates charts and insights
- `students_cleaned.csv` → cleaned output after running the cleaning step
- `outputs/` → charts, summary statistics, and insights

## How to Run
1. Install requirements:
   ```bash
   pip install pandas matplotlib
   ```

2. Run the cleaning script:
   ```bash
   python clean_data.py
   ```

3. Run the analysis script:
   ```bash
   python analyze_data.py
   ```
